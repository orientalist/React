# 建立環境
> [本篇來源](https://reactjs.org/docs/create-a-new-react-app.html#create-react-app)
## Node.js
Node >= 8.10
***
## npm
npm >= 5.6
***
## 執行指令
先切換至要初始化react專案的資料夾,輸入
```terminal
npx create-react-app 專案名
```
## 執行react專案
```terminal
npm start
```
## 發布
於專案資料夾下建立build資料夾,內含發布後的檔案
```terminal
npm run build
```
